# WebGAL Export Demo

This folder mirrors the asset layout expected by a WebGAL game package.

- `game/scene/start.txt`: generated story script for the demo answer.
- `game/background/`: background assets used by `changeBg`.
- `game/figure/`: character assets used by `changeFigure`.

The static prototype in the repo renders its own lightweight preview shell, while this export folder is the handoff target for wiring the same content into a real OpenWebGAL runtime.
